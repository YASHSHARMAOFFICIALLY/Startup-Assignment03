# Brand — SalesIO OS

_Status: active_

## Palette

All colors are defined as Tailwind tokens under `brand.*` in `tailwind.config.ts`.

| Token | Hex | Usage |
|-------|-----|-------|
| `brand-bg` | `#0A0A0B` | Page background |
| `brand-surface` | `#141517` | Sidebar, topbar, card backgrounds |
| `brand-elevated` | `#1F2024` | Active nav, hover panels |
| `brand-border` | `#26272B` | Borders |
| `brand-accent` | `#F59E0B` | Primary accent (gold) — CTAs, charts, focus rings |
| `brand-teal` | `#00E5C4` | Secondary accent — sync buttons, offers CTAs |
| `brand-purple` | `#A855F7` | Tertiary accent — DM setter icons |
| `brand-textPrimary` | `#FFFFFF` | Primary text |
| `brand-textSecondary` | `#D4D4D8` | Secondary text |
| `brand-textMuted` | `#A1A1AA` | Muted text |
| `brand-textFaint` | `#71717A` | Faint/disabled text |
| `brand-positive` | `#22C55E` | Positive deltas, success |
| `brand-negative` | `#EF4444` | Negative deltas, errors |
| `brand-gold` | `#F59E0B` | Leaderboard rank 1 |
| `brand-silver` | `#C0C5CE` | Leaderboard rank 2 |
| `brand-bronze` | `#A9744F` | Leaderboard rank 3 |
| `brand-iconBg` | `#0D1117` | Icon container backgrounds |
| `brand-colHeader` | `#8B9AB3` | Table column header text |

## Typography

- **Sans:** Geist Sans (`--font-geist-sans`) via `next/font`
- **Mono:** Geist Mono (`--font-geist-mono`) via `next/font`

## Selection

- Background: `brand-accent` (gold)
- Text: black

## Voice

- Concise, active voice
- Professional but not corporate
- "Sales.io OS" — always with the period after "Sales"
